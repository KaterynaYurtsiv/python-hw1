# Python Homework 1

## Description
This program works with a 3x3 two-dimensional array of random integers from 1 to 100.

The program provides several operations through a Tkinter interface:
- create a random 3x3 array
- display the array
- calculate the sum of all elements
- find the maximum and minimum values with their indices
- sort each row of the array
